# jnaOTheme

jnaOTheme is a dark theme for sublime text. Based on the default theme, some new icons, retina icons and new colors.

## Installation

* Go to sublime packages folder
* Clone this repo
* in sublime settings, add `"theme": "jnaOTheme.sublime-theme"`
* Save
* Restart sublime
